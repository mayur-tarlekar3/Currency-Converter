# Currency Converter 💱

A simple **Currency Converter Web App** built using **HTML, CSS, and JavaScript**.
This app allows users to convert money from one currency to another using real-time exchange rates.

## 🚀 Features

* Convert currency between different countries
* Shows country flags
* Uses exchange rate API
* Simple and responsive design

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
* Font Awesome
* Exchange Rate API

## 📂 Project Structure

currency-converter
│
├── index.html
├── style.css
├── app.js
├── codes.js

## ▶️ How to Use

1. Enter the amount.
2. Select the **From** currency.
3. Select the **To** currency.
4. Click **Get Exchange Rate**.
5. The converted amount will be displayed.
